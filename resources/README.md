# Recursos de cubiApp

Coloca aquí el **icono/logo** de la aplicación:

- **icon.png** o **logo.png** — la app lo usará como icono en la barra de título y en el Dock (macOS).

También puedes poner el archivo en la raíz del proyecto (`ProyectoJose/icon.png` o `ProyectoJose/logo.png`).

Formatos admitidos: PNG, ICO (Windows), ICNS (macOS).
